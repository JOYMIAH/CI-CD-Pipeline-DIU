About DevOps Engineering: DevOps & DevSecOps is a set of practices that combines software development (Dev) and System Security (sec) and IT operations (Ops). It aims to shorten the systems development life cycle and provide continuous delivery with high software quality.

Technology Used:

  • Virtuaization(KVM): A technology that allows running multiple operating systems on a single hardware host.
  • Linux Server's: A version of the Linux operating system designed to handle the more demanding needs of business applications such as network and system administration, database management and web services.
  • Jenkins: An open-source automation tool written in Java with plugins built for Continuous Integration purposes.
  • Docker: An open-source platform used for containerization.
  • Kubernetes: An open-source platform designed to automate deploying, scaling, and operating application containers.
  • SonarQube: An open-source platform developed by SonarSource for continuous inspection of code quality.
  • Trivy: A simple and comprehensive vulnerability scanner for containers.
  • ArgoCD: A declarative, GitOps continuous delivery tool for Kubernetes.
  • Helm: The package manager for Kubernetes.
  • Promethus: An open-source systems monitoring and alerting toolkit.
  • Altert-Manager: Handles alerts sent by client applications such as the Prometheus server.
  • Grafana: An open-source platform for monitoring and observability.
